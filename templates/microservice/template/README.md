## Observability

The template includes a local `OpenTelemetry Collector` config at `deploy/otel-collector.yaml`.

When you run `make up`, the service exports telemetry to the collector container. When you run `make run`, the service uses the endpoint from `configs/config.yaml` unless overridden by `OTEL_EXPORTER_OTLP_ENDPOINT`.
