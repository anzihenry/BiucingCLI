## Execution Modes

- `scheduled`: runs the starter task immediately and then on a repeating interval
- `oneshot`: runs the starter task once and exits

The generated defaults are:

- `run_mode={{RUN_MODE}}`
- `tick_interval_seconds={{TICK_INTERVAL_SECONDS}}`
- `shutdown_timeout_seconds={{SHUTDOWN_TIMEOUT_SECONDS}}`

You can override them with environment variables:

- `WORKER_NAME`
- `WORKER_RUN_MODE`
- `WORKER_TICK_INTERVAL_SECONDS`
- `WORKER_SHUTDOWN_TIMEOUT_SECONDS`
- `CONFIG_FILE`

Example oneshot run:

```bash
WORKER_RUN_MODE=oneshot make run
```

Example scheduled run with a shorter loop:

```bash
WORKER_RUN_MODE=scheduled WORKER_TICK_INTERVAL_SECONDS=5 make dev
```
